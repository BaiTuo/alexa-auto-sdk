Detailed Description

The MediaPlayer class is the base class for platform media players, and should be extended to handle them.

◆ isRepeating()

boolean com.amazon.aace.alexa.MediaPlayer.isRepeating ( )

Returns true if the current media source should repeat when it is finished playing. The MediaPlayer is responsible for repeating the current audio if this call returns true. If false is returned, the MediaPlayer should call playbackFinished() when the current audio is done playing.

Returns
true if the call was handled successfully.
